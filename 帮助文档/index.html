Introduction


Martyn CV - Resume / CV / Portfolio & vCard HTML5 Template

Creating a professional, modern and sophisticated online resume website is easy with Martyn CV / Resume. Martyn CV / Resume HTML Template is a perfect solution for professional resume / cv / portfolio and personal website. it comes with a lot of useful features. It is easy to customize this template for any of Your website needs.

It comes in dark and light modes and you can also pick between different predefined samples. There is a lot of opportunities you unlock yourself once you unbox Martyn CV / Resume.

Configuration and Installation

(1) Martyn-CV-HTML and (2) Documentation folder

Open the "Martyn-CV-HTML" folder and run index.html in your browser

  1. Open the Martyn-CV-HTML Folder to find all the Templates Files

    • Martyn-CV-HTML/assets/js - Javacripts Folder
    • Martyn-CV-HTML/assets/owlcarousel - owlcarousel Stylesheet File
    • Martyn-CV-HTML/assets/index.html - Index File/Homepage
    The other files can be used according to your preferences.
  4. You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

FAQ



How to remove the top menu sticky as scrolling?

  • Open Up your HTML file 'Martyn CV-HTML/' file location. (For Example, 'index.html'
  • Then Search For the Comment '<!-- START HEADER -->'
  • show this line - <header class="header_wrap">
  • add class name - <header class="header_wrap no-sticky"> (add class name 'no-sticky')

Where i can define my e-mail account to receive the messages from contact form?

  • Open Up your HTML file 'Martyn CV-HTML' file location. (For Example, 'Martyn CV-HTML/contact.php'
  • show this line number 11 - $mailTo = 'email@example.com'
  • 'email@example.com' enter your email account to receive the messages from contact form
